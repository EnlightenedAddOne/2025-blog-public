# 运行安装宝塔命令

- 连接好树莓派后，在终端输入下面指令

```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec
```

- 根据提示选择 _y_ ，等待程序执行
- 由于我使用的树莓派3B，并且网络环境一般，程序执行较慢（70分钟）
- 安装完成后在终端会显示宝塔面板的*ip*以及*账号密码*

# 登录宝塔

在浏览器里输入终端显示的IP进入宝塔登录界面，输入账号密码进入。

进入之后根据提示注册宝塔账号，安装基础环境（安装极慢），由于在安装 _MySQL_ 时出错，显示运行内存较小，所以安装了较老版本的MySQL
