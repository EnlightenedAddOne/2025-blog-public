# 武汉理工大学 计算机考研详档（2026版）

- 地区：湖北武汉
- 院校层次：211 / 双一流 / 教育部直属

## 3. 复试与录取规则

- 复试形式：线下现场复试
  - 笔试（100分）：算法设计与分析、程序设计、离散结构
  - 面试（100分）：外语自述、抽题作答、专家提问
- 复试占比：初试70% + 复试30%，总成绩=初试成绩/5×70%+复试成绩×30%
- 同等力学加试要求：加试数据库、软件工程（笔试），及格线60分
- 调剂政策：详见学院复试细则

## 4. 导师与研究方向

- 导师检索入口：http://cse.whut.edu.cn/（计算机学院官网）
- 代表研究方向：
  - 智能方法与智能系统
  - 大数据理论及应用
  - 高性能计算与云计算
  - 网络与信息安全技术
- 学科平台：交通物联网技术湖北省重点实验室、华为ICT学院、"智能基座"产教融合基地

## 5. 学费与奖助体系

- 学费标准（全日制）：
  - 学术型硕士：8000元/生·学年
  - 专业学位硕士：8000元/生·学年
- 国家助学金：6000元/生·学年（全日制非定向）
- 学业奖学金：
  - 一等奖学金：12000元/生·学年（比例20%）
  - 二等奖学金：8000元/生·学年（比例30%）
  - 三等奖学金：4000元/生·学年（比例40%）
- 其他专项资助：三助岗位、国家奖学金、专项奖学金、企业奖学金
