# 指令及其含义

> 来源：Obsidian/10-学业与考试/期末复习资料/嵌入式/指令及其含义.md

这是最基础且核心的部分，重点掌握 `LDR`, `STR`, `MOV`, `ADD`, `B/BL`。

1. **⭐ `ADD R0, R0, #1`**: 加法。`R0 = R0 + 1`。

2. **⭐ `ADD R0, R1, R2`**: 加法。`R0 = R1 + R2`。

3. **⭐ `LDR R0, [R1]`**: 加载字 (Load Word)。从 R1 指向的内存地址读取数据存入 R0。

4. **⭐ `STR R0, [R1]`**: 存储字 (Store Word)。将 R0 的数据写入 R1 指向的内存地址。

5. **⭐ `LDR R0, [R1, #4]`**: 前索引偏移加载。从地址 `(R1 + 4)` 读取数据到 R0。R1 本身不变。

6. `LDR R0, [R1], #4`: 后索引加载。先从 R1 地址读取数据到 R0，然后 `R1 = R1 + 4`。常用于遍历数组。

7. `LDR R0, [R1, #4]!`: 前索引回写加载。先计算 `R1 = R1 + 4`，然后从新地址读取数据到 R0。

8. `LDR R0, [R1, R2]`: 寄存器偏移加载。读取地址 `(R1 + R2)` 的数据。

9. `LDMIA R0, {R1, R2, R3, R4}`: 多寄存器加载 (Load Multiple Increment After)。从 R0 指向的地址开始，依次读取数据填充到 R1-R4。

10. `LDMDB R0, {R1, R2, R3, R4}`: 多寄存器加载 (Decrement Before)。地址先递减再读取。

11. **⭐ `MOV R0, R1, LSL #2`**: 移位传送。`R0 = R1 << 2` (左移2位，相当于乘以4)。

12. **⭐ `STMFD R13!, {R0, R4-R12, LR}`**: 入栈 (Push)。将 R0, R4-R12 和 LR (返回地址) 保存到堆栈（R13/SP）。常用于**函数开头**保护现场。

13. **⭐ `LDMFD R13!, {R0, R4-R12, PC}`**: 出栈 (Pop)。从堆栈恢复寄存器，并将之前的 LR 恢复给 PC (程序计数器)，实现**函数返回**。

14. `LDR R0, [R1, R2, LSL #2]!`: 复杂的带移位前索引回写加载。

15. `LDR R0, [R1], R2, LSL #2`: 复杂的带移位后索引加载。

16. `STR R0, [R1], #8`: 后索引存储。存完数据后，`R1 = R1 + 8`。

17. **⭐ `B loop A`**: 无条件跳转 (Branch)。跳转到标签 `loop A` 处执行。

18. **⭐ `BL sub`**: 带链接跳转 (Branch with Link)。跳转到 `sub` 子程序，并将下一条指令地址保存到 LR (R14)。用于**函数调用**。

19. **⭐ `BX R6`**: 交换跳转 (Branch and Exchange)。跳转到 R6 中的地址，并可能切换指令集状态（ARM/Thumb）。

20. `BLX R0`: 带链接和交换跳转。调用 R0 地址的函数并切换状态。

21. `ADD R2, R1, R3`: 加法。`R2 = R1 + R3`。

22. **⭐ `SUBS R8, R6, #240`**: 减法并更新标志位 (Subtract and Set flags)。`R8 = R6 - 240`，结果会影响 CPSR 状态寄存器（如结果为0，Z位置1）。

23. `RSB R4, R4, #1280`: 逆向减法 (Reverse Subtract)。`R4 = 1280 - R4`。

24. `ADCHI R11, R0, R3`: 带进位加法 (Add with Carry)，仅在条件 HI (无符号大于) 满足时执行。

25. `RSCLES ...`: 复杂的条件逆向带进位减法。

26. `RSCLES ...`: 同上，操作数为 PC (R15)。

27. **⭐ `AND R9, R2, #0xFF00`**: 按位与。`R9 = R2 & 0xFF00`。常用于掩码操作（取某些位）。

28. `ORREQ R2, R0, R5`: 按位或 (OR)，仅在条件 EQ (相等) 时执行。

29. `EORS R0, R0, R3, ROR R6`: 异或并更新标志位。

30. `BICNES R8, R10, R0, RRX`: 位清除 (Bit Clear)，条件为 NE，更新标志位 S。逻辑为 `R8 = R10 AND NOT (R0 带扩展旋转)`。

31. `EORS R0, R15, R3, ROR R6`: 异或 (XOR)，更新标志位。

32. **⭐ `MOV R5, R2`**: 数据传送。`R5 = R2`。

33. `MVNNE R11, #0x0F0`: 数据取反传送 (Move Not)，条件为 NE。`R11 = ~0x0F0`。

34. `MOVS R0, R0, ASR R3`: 算术右移传送，更新标志位。

35. **⭐ `CMP R2, R9`**: 比较 (Compare)。执行 `R2 - R9`，但不保存结果，只更新标志位 (CPSR)。用于 `if` 判断。

36. `CMN R0, #6400`: 负数比较 (Compare Negative)。执行 `R0 + 6400` 更新标志位。

37. `CMPGT R13, R7, LSL #2`: 条件比较。如果上一步结果是 GT (大于)，则进行比较。

38. `CMP R2, R15, ASR R0`: 比较指令，第二个操作数带移位。

39. **⭐ `TST R0, #0x3F8`**: 测试位 (Test)。执行 `R0 & 0x3F8`，不保存结果，只更新标志位。常用于检查特定位是否为1。

40. `TEQEQ R10, R9`: 测试等价 (Test Equivalence)，也就是异或测试，条件为 EQ。

41. `TSTNE R1, R5, ASR R1`: 条件测试。

42. `TEQ R15, R1, ROR R0`: 测试等价。

43. **⭐ `MUL R10, R2, R5`**: 乘法。`R10 = R2 * R5` (32位结果)。

44. `MLA R10, R2, R1, R5`: 乘加 (Multiply Accumulate)。`R10 = (R2 * R1) + R5`。

45. `MULS R0, R2, R2`: 乘法并更新标志位。

46. `MULLT R2, R3, R2`: 条件乘法 (LT: 小于)。

47. `MLAVCS ...`: 条件乘加。

48. `MUL R15, R0, R3`: 结果写入 PC (R15)，这会导致跳转（通常不建议这样写，除非故意为之）。

49. `SMULL R0, R1, R2, R3`: 有符号长乘法 (64位结果)。`R1:R0 = R2 * R3` (R1存高32位，R0存低32位)。

50. `SMLAL ...`: 有符号长乘加。

51. `UMULL ...`: 无符号长乘法。

52. `UMLAL ...`: 无符号长乘加。
