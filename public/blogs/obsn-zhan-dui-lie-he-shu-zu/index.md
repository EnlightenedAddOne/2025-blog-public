# 第三章：栈、队列和数组

## 1. 栈 (Stack)

- 对应视频：`3.1.1_栈的基本概念`、`3.1.2_栈的顺序存储实现`
- **定义**：只允许在一端（栈顶 Top）进行插入和删除的线性表。
- **特点**：**后进先出 (LIFO)**。
- **存储**：
  - 顺序栈：`Top` 指针（注意是指向当前元素还是下一个空位）。判满/判空条件。
  - 链栈：通常无头结点的单链表，头插法。
- **应用**：
  - 对应视频：`3.3.1_栈在括号匹配中的应用`、`3.3.2_栈在表达式求值中的应用`、`3.3.3_栈在递归中的应用`
  - **括号匹配**。
  - **表达式求值**：中缀转后缀（逆波兰），后缀表达式计算。
  - **递归**：函数调用栈（斐波那契、汉诺塔）。

## 2. 队列 (Queue)

- 对应视频：`3.2.1_队列的基本概念`、`3.2.2_队列的顺序实现`
- **定义**：一端（队尾 Rear）插入，另一端（队头 Front）删除。
- **特点**：**先进先出 (FIFO)**。
- **存储**：
  - **循环队列**（重点）：利用取模运算 `%` 实现逻辑环状。
    - 队空：`front == rear`
    - 队满：`(rear + 1) % MaxSize == front` (牺牲一个单元法)
    - 元素个数：`(rear - front + MaxSize) % MaxSize`
    ```c
    // 入队操作
    bool EnQueue(SqQueue &Q, ElemType x) {
        if((Q.rear + 1) % MaxSize == Q.front) return false; // 队满
        Q.data[Q.rear] = x;
        Q.rear = (Q.rear + 1) % MaxSize;
        return true;
    }
    ```
  - **链队列**：带头结点的单链表，`front` 指头，`rear` 指尾。
- **双端队列 (Deque)**：
  - 对应视频：`3.2.4_双端队列`
  - 两端都可入可出。考点：判断输出序列的合法性。

  - **对称矩阵**：只存上三角或下三角，$a_{ij}$ 与 $k$ (数组下标) 的映射关系。
  - **三角矩阵**。
  - **三对角矩阵**（带状矩阵）。
  - **稀疏矩阵**：三元组表、十字链表。
