# 第二章：线性表

## 1. 线性表的定义

- 具有相同数据类型的 $n$ ($n \ge 0$) 个数据元素的有限序列。
- 特点：除第一个元素外，每个元素有且仅有一个直接前驱；除最后一个元素外，每个元素有且仅有一个直接后继。

## 2. 顺序表 (Sequential List)

- **实现**：数组（静态分配或动态分配 `malloc`）。
- **特点**：
  - **随机访问**：$O(1)$ 时间通过下标访问。
  - **存储密度高**。
  - **插入/删除**：需要移动大量元素，平均时间复杂度 $O(n)$。
- **重要操作**：
  - 插入：平均移动 $n/2$ 个元素。
  - 删除：平均移动 $(n-1)/2$ 个元素。

## 3. 链表 (Linked List)

- **单链表**：
  - **头指针 vs 头结点**：头结点方便操作（插入/删除第一个元素时逻辑统一），头指针指向链表起始。
  - **建立**：
    - **头插法**：读入顺序与生成顺序相反（可用于链表逆置），$O(n)$。
    - **尾插法**：读入顺序与生成顺序相同，需尾指针，$O(n)$。
- **双链表**：
  - 节点含 `prior` 和 `next` 指针。
  - 删除/插入操作需修改前后两个方向的指针（注意判空）。
- **循环链表**：
  - 循环单链表：`rear->next == head`。判空条件：`head->next == head` 或 `rear == head`（若有尾指针）。
  - 循环双链表。
- **静态链表**：用数组模拟链表，指针是数组下标（游标）。

## 4. 顺序表 vs 链表

| 特性              | 顺序表                | 链表                             |
| :---------------- | :-------------------- | :------------------------------- |
| **存取方式**      | 随机存取              | 顺序存取                         |
| **逻辑/物理结构** | 逻辑相邻物理也相邻    | 逻辑相邻物理不一定相邻           |
| **插入/删除**     | 慢 $O(n)$，需移动元素 | 快 $O(1)$ (已知位置)，只需改指针 |
| **空间分配**      | 易产生碎片或溢出      | 动态分配，灵活                   |

## 5. 核心代码题型

1. **逆置**：原地逆置顺序表或链表。
   ```c
   // 链表原地逆置（头插法）
   void Reverse(LinkList &L) {
       LNode *p = L->next, *r;
       L->next = NULL;    // 将头结点摘下
       while(p != NULL) { // 依次将节点插入头结点之后
           r = p->next;   // 暂存后继
           p->next = L->next;
           L->next = p;
           p = r;
       }
   }
   ```
2. **删除**：删除特定值、删除重复值、删除区间值。
   ```c
   // 顺序表删除最小值（假设唯一）
   bool DelMin(SqList &L, ElemType &value) {
       if(L.length == 0) return false;
       value = L.data[0];
       int pos = 0;
       for(int i=1; i<L.length; i++) {
           if(L.data[i] < value) {
               value = L.data[i];
               pos = i;
           }
       }
       L.data[pos] = L.data[L.length-1]; // 用最后一个元素填补
       L.length--;
       return true;
   }
   ```
3. **合并**：合并两个有序表为新的有序表（归并思想）。
4. **快慢指针**：找链表中间节点、倒数第 k 个节点、判断是否有环。
   ```c
   // 查找链表中间节点
   LNode* FindMid(LinkList L) {
       LNode *fast = L, *slow = L;
       while(fast != NULL && fast->next != NULL) {
           fast = fast->next->next; // 快指针走两步
           slow = slow->next;       // 慢指针走一步
       }
       return slow; // slow即为中间节点
   }
   ```
