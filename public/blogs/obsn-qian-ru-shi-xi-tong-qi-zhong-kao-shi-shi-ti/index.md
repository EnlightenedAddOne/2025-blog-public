#### 洛阳理工学院 2025/2026 学年 第一学期 嵌入式系统 期中考试试题

**一、选择题 (每小题2分，共 10 分)**

1. ARM 嵌入式处理器中，以下哪个不是 ARM 体系结构的特征 ( )。

   A. 精简指令集

   B. 支持多核处理器

   C. 多种寄存器

   D. CISC 架构

2. 寄存器 R15 除了可以做通用寄存器外，还可以做 ( )。

   A. 程序计数器

   B. 基址寄存器

   C. 链接寄存器

   D. 堆栈指针寄存器

3. 存储一个 32 位数 0x87654321 到 1000H~1003H 四个字节单元中，若以小端格式存储，则 1000H 的存储单元的内容为 ( )。

   A. 0x12

   B. 0x21

   C. 0x78

   D. 0x87

4. ARM 汇编语句 “EOR R0, R1,#0xFF” 若 R1=0x12345678, 则运行后 R0 为 ( )。

   A. 0x123456FF

   B. 0x12345687

   C. 0x12345678

   D. 0x00000078

5. ARM 汇编语句 “STR R0, \[R1, R2, LSR#1]” 若 R1=0x1000, R2=0x11, 则运行后 R0 被保存到的地址为 ( )。

   A. 0x1000

   B. 0x1001

   C. 0x1010

   D. 0x1008

5. ARM920T 体系结构所支持的最大寻址空间为 ( 10 ) G；其支持的两种存储字节数据的格式分别是 ( 11 ) 格式和 ( 12 ) 格式，其中在 ( 13 ) 格式中，数据的低字节存放在高地址单元中，而字数据的高字节存放在低地址单元中。

**四、程序分析题 (每空2分，共 18 分)**

1. 对下列指令进行注释。
   - `ADD R0, R1, R2` ; ( 1 )
   - `LDMIA R0, {R1, R2, R3, R4}` ; ( 2 )

   - `MLA R10, R2, R1, R5` ; ( 5 )
   - `UMULL R0, R1, R2, R3` ; ( 6 )

   ```
       ADD R2, R0, R1
       SUB R0, R0, #1
       CMP R0, #0
       BNE Loop
   ```

   写出完成读指令后，R0 的值为 0x ( ) **\_\_**，R1 的值为 0x ( ) **\_\_**，R2 的值为 0x ( ) **\_\_**。
