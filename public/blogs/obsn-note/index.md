# 📋 目前可用的传感器列表

根据你的代码，以下是**所有可用的传感器**及其功能：

---

## ✅ 1. **温湿度传感器**

**文件：** tah.c

| 函数         | 返回值 | 单位 | 说明     |
| ------------ | ------ | ---- | -------- |
| `temp_get()` | float  | °C   | 环境温度 |
| `hum_get()`  | float  | %    | 相对湿度 |

**硬件路径：**

```c
/sys/bus/iio/devices/iio:device0/in_temp_input       // 温度
/sys/bus/iio/devices/iio:device0/in_humidityrelative_input  // 湿度
```

**测试状态：** ✅ 已集成到 MQTT 上报

---

## ✅ 2. **光照传感器**

**文件：** light.c

| 函数             | 返回值 | 单位 | 说明                 |
| ---------------- | ------ | ---- | -------------------- |
| `illumi_get()`   | float  | lux  | 环境光照强度         |
| `infrared_get()` | float  | -    | 红外光强度（原始值） |

**硬件路径：**

```c
/sys/bus/iio/devices/iio:device1/in_illuminance_input     // 可见光
/sys/bus/iio/devices/iio:device1/in_illuminance_ir_raw    // 红外光
```

**测试状态：** ✅ 已集成到 MQTT 上报

---

## ✅ 3. **人体红外传感器（接近传感器）**

**文件：** light.c

| 函数             | 返回值 | 单位 | 说明                       |
| ---------------- | ------ | ---- | -------------------------- |
| `distance_get()` | float  | -    | 接近度原始值               |
| `human_detect()` | int    | -    | 人体检测（1=有人，0=无人） |

**硬件路径：**

```c
/sys/bus/iio/devices/iio:device1/in_proximity_raw
```

**阈值：** `INFRARED_THRESHOLD = 50.0`
**逻辑：**

```c
if (distance > 50.0) → 有人 (1)
else → 无人 (0)
```

**测试状态：** ✅ 已集成到 MQTT 上报（状态变化时上报）

---

## ✅ 4. **电压传感器**

**文件：** voltage.c

| 函数            | 返回值 | 单位 | 说明     |
| --------------- | ------ | ---- | -------- |
| `voltage_get()` | float  | V    | 系统电压 |

**硬件路径：**

```c
/sys/bus/iio/devices/iio:device3/in_voltage1_raw      // 原始ADC值
/sys/bus/iio/devices/iio:device3/in_voltage_offset    // 偏移校准
/sys/bus/iio/devices/iio:device3/in_voltage_scale     // 比例系数
```

**计算公式：**

```c
voltage = (raw + offset) * scale / 1000
```

**测试状态：** ✅ 测试通过，未集成到 MQTT

# 🏠 智能家居传感器联动方案

基于现有硬件，设计一个**自动化智能家居系统**：

---

## 🎯 联动场景设计

### **场景1：智能照明系统**

**触发条件：** 光照 < 20 lux **且** 检测到人体
**执行动作：** 自动开灯

**场景2：智能安防系统**
**触发条件：** 检测到人体 **且** 光照 < 10 lux（夜间）
**执行动作：** 蜂鸣器报警 + LED闪烁

**场景3：智能通风系统**
**触发条件：** 温度 > 28°C **或** 湿度 > 70%
**执行动作：** 自动开启风扇

**场景4：离开模式**
**触发条件：** 无人检测超过 30 秒
**执行动作：** 关闭所有灯光和风扇（节能）

**场景5：环境异常报警**
**触发条件：** 温度 > 35°C **或** 电压异常
**执行动作：** 蜂鸣器持续报警

---

## 📊 联动场景总结表

| 场景                | 触发条件              | 执行动作             | 优先级 |
| ------------------- | --------------------- | -------------------- | ------ |
| 🔴 **环境异常报警** | 温度>35°C 或 电压异常 | 高频蜂鸣器持续报警   | 最高   |
| 🚨 **智能安防**     | 夜间(光<10lux) + 有人 | 蜂鸣器报警 + LED闪烁 | 高     |
| 💡 **智能照明**     | 光<20lux + 有人       | 自动开灯             | 中     |
| 🌀 **智能通风**     | 温度>28°C 或 湿度>70% | 启动风扇             | 中     |
| 💤 **节能模式**     | 无人超过30秒          | 关闭灯光+风扇        | 低     |

---

## 🎮 云端控制扩展

可以通过 MQTT 远程配置联动开关：

```json
// 关闭自动照明
{"config":"auto_light","value":false,"id":0}

// 调整风扇阈值
{"config":"temp_threshold","value":25.0,"id":0}

// 启用节能模式
{"config":"energy_saving","value":true,"id":0}
```
