## 1. 基础概念与数学工具

### 1.1 概率与矩阵

- **泊松分布**：常用于描述一定时间内 **随机事件发生次数** 的概率分布。
- **矩阵运算**：
  - 规则：$A(m \times n) \times B(n \times p) = C(m \times p)$。
  - 记忆：中间维度必须相同并抵消，保留首尾维度。

### 1.2 信息论基础

> [!abstract] 核心定义
>
> - **消息 (Message)**：信息的 **具体表现形式**（如文字、图像）。
> - **信息 (Information)**：消息的 **内涵**（消除不确定性的度量）。
> - **信号 (Signal)**：消息的 **物理载体**（如电信号、光信号）。

- **信息量规律**：
  - 信息量是概率的单调递减函数。**概率越小，信息量越大**。
  - 单位：**比特 (bit)**。

### 1.3 信号类型

| 类型         | 特征                              | 优缺点                               |
| :----------- | :-------------------------------- | :----------------------------------- |
| **模拟信号** | 时间和幅度均 **连续**             | 结构简单但抗噪差                     |
| **数字信号** | 时间和幅度均 **离散**（有限个值） | 抗干扰强，易加密存储，但不一定省带宽 |

## 2. 必背计算公式

### 2.1 平均信息量（熵）

$$H(X) = - \sum_{i=1}^{n} p_i \log_2 p_i$$

> [!example] 例题
> 4个符号概率分别为 $1/4, 1/8, 1/8, 1/2$。
> **计算**：
> $H = -( \frac{1}{4}\log_2\frac{1}{4} + \frac{1}{8}\log_2\frac{1}{8} + \frac{1}{8}\log_2\frac{1}{8} + \frac{1}{2}\log_2\frac{1}{2} )$
> $H = 0.5 + 0.375 + 0.375 + 0.5 = 1.75 \text{ bit/symbol}$

### 2.2 香农公式（信道容量）

$$C = B \log_2(1 + \frac{S}{N})$$

- **$B$**：信道带宽 (Hz)
- **$S/N$**：信噪比 (注意题目给的是分贝还是倍数)
- **$C$**：信道容量 (bit/s)

> [!tip] 考试技巧
> 若 $B=3.4\text{kHz}$，信噪比 $S/N=1000$：
> $C = 3400 \times \log_2(1001) \approx 3400 \times 10 \approx 34000 \text{ bit/s}$。

## 3. 调制技术 (重点)

### 3.1 模拟调制

- **AM (常规调幅)**：
  - 包络形状与调制信号波形一致。
  - 大信噪比用 **包络检波**（简单），小信噪比有 **门限效应**。
- **DSB (抑制载波双边带)**：
  - 抑制了载波，**不能** 用包络检波，必须用 **相干解调**。
  - 带宽 $= 2 \times$ 基带带宽。
- **SSB (单边带)**：
  - **最省带宽**：带宽 $= 1 \times$ 基带带宽。
  - 相比DSB最主要优点：**节省了一半带宽**。
- **VSB (残留边带)**：SSB和DSB的折中。
- **FM (调频) / PM (调相)**：
  - 属于 **非线性调制**。
  - 核心优势：**抗噪声性能好**（以牺牲带宽为代价）。

### 3.2 数字调制

- **2ASK(二进制振幅键控)**：抗干扰性能最差。
- **2PSK(二进制相移键控)**：
  - 频带利用率高，抗幅度起伏。
  - 缺点：**相位模糊**，不能用直接法提取载波。
- **2DPSK**：解决相位模糊问题。
- **抗噪性能排序** (相干解调)：$2PSK > 2FSK > 2ASK$ (2ASK最差)。
- **QAM**：幅度和相位结合，星座点间距大，误码率低。

### 3.3 系统指标对比

- **模拟系统**：
  - 有效性 $\rightarrow$ **占用带宽**。
  - 可靠性 $\rightarrow$ **输出信噪比**。
- **信道编码**：主要目的是提高 **可靠性**。

## 4. 模数转换 (ADC)

过程：**抽样 $\rightarrow$ 量化 $\rightarrow$ 编码**

1. **抽样 (Sampling)**：
   - **奈奎斯特采样定理**：$f_s \ge 2 f_{max}$。
   - **混叠 (Aliasing)**：若 $f_s < 2 f_{max}$，高频会折叠到低频，无法恢复。
   - 恢复信号：使用 **低通滤波器**。
2. **量化 (Quantization)**：
   - **唯一引入失真** 的步骤（量化噪声）。
   - **非均匀量化**：改善 **小信号** 的信噪比。
3. **编码 (Encoding)**：
   - 变长编码（霍夫曼）：按概率分配码长（概率大用短码），目的是降低平均码长。

## 5. 先进通信技术

### 5.1 扩频与多载波

- **扩频通信 (Spread Spectrum)**：
  - 特征：信号带宽 $\gg$ 信息带宽。
  - 典型应用：**蓝牙** (FH-SS/DS-SS)。
- **OFDM**：
  - 核心：子载波相互 **正交**，并行传输。
  - 缺点：对 **载波频偏** 和相位噪声非常敏感。

### 5.2 MIMO 与 UWB

- **MIMO**：
  - 利用多天线实现 **空间复用**（提速）和 **空间分集**（抗衰落）。
- **UWB (超宽带)**：
  - 特征：极窄脉冲，**功率谱密度极低**，带宽极宽。
  - 场景：高精度定位，**不适合** 远距离卫星通信。

## 6. 网络协议与架构

### 6.1 编码类型

| 类型         | 功能                             | 例子          |
| :----------- | :------------------------------- | :------------ |
| **信源编码** | **压缩** 数据，提高有效性        | 霍夫曼编码    |
| **信道编码** | 增加冗余，提高 **可靠性** (纠错) | 前向纠错(FEC) |

- **汉明距离**：最小距离 $d_{min}=3$ 时，可 **检2位错** 或 **纠1位错**。

### 6.2 MAC 与 路由

- **CSMA/CA**：用于无线（WiFi），核心是 **冲突避免**。
- **距离向量路由 (DV)**：容易产生 **“计数到无穷”** 问题（由环路引起）。
- **链路状态路由 (LS)**：收敛快，无环路。

### 6.3 网络类型

- **Ad Hoc (自组织网)**：
  - 无基础设施，无中心，多跳。
  - 节点角色：既是主机又是路由器。
- **WSN (无线传感器网)**：
  - 首要设计目标：**最大化网络生命周期**（极致节能）。

## 7. 易混淆点速记 (Flashcards)

**Q: 蜂窝通信为什么要频率复用？**
A: 因为频谱有限。通过在相隔足够远的小区重复使用相同频率，提高系统容量。

**Q: 单边带 (SSB) 的带宽是多少？**
A: 等于基带信号带宽（例如基带21kHz，SSB也是21kHz）。

**Q: 哪种编码步骤会引入不可消除的失真？**
A: 量化。

**Q: 2PSK 为什么不能用直接法提取载波？**
A: 存在 $180^\circ$ 相位模糊。

**Q: 物联网 MAC 协议最优先考虑什么？**
A: 节能（能量效率）。

### 1. 模拟调制技术的比较 (AM / DSB / SSB / FM)

| **调制方式**       | **核心特点**         | **带宽占用**     | **抗噪声性能** | **备注/考点**                                                                     |
| ------------------ | -------------------- | ---------------- | -------------- | --------------------------------------------------------------------------------- |
| **AM (常规调幅)**  | 结构最简单，包络检波 | $2f_m$ (双边带)  | 较差           | 优点是**解调简单、成本低**；缺点是功率利用率低，且小信噪比下有**门限效应** 。<br> |
| **DSB (双边带)**   | 抑制了载波           | $2f_m$ (双边带)  | 一般           | **不能**用包络检波（因为没有载波分量），必须用复杂的相干解调 2222。<br>           |
| **SSB (单边带)**   | 滤除了一半边带       | **$f_m$ (最小)** | 一般           | **最省带宽**（频带利用率最高），但滤波器设计困难 。                               |
| **VSB (残留边带)** | 折中方案             | $\approx f_m$    | 一般           | 介于SSB和DSB之间，常用于**电视广播**（能保留低频分量）。                          |
| **FM (调频)**      | 非线性调制           | **最宽**         | **最好**       | 以**牺牲带宽**为代价换取高抗噪声性能（优于AM）。                                  |

- **总结记忆：**
  - **最省带宽**：SSB 。
  - **抗噪最好**：FM 。
  - **最简单**：AM 。
  - **非线性调制**：FM/PM（频谱不仅仅是搬移，还产生了新频率）。

---

### 2. 数字调制技术的比较 (ASK / FSK / PSK / QAM)

这部分重点考察**抗干扰能力（误码率）** 和 **频带利用率**的排序。

#### (1) 抗噪声性能（误码率）排序

在相同的信噪比下，谁最不容易出错？

- **排序（从优到差）：** **2PSK > 2DPSK > 2FSK > 2ASK** 。
  - **2PSK (相移键控)**：**性能最好**，对信道幅度变化不敏感，频带利用率高，但需要相干解调 。
  - **2ASK (幅移键控)**：**性能最差**，对信道特性变化最敏感（因为信息在幅度里，容易被干扰）。

#### (2) 频带利用率（带宽效率）

谁能传输得更快、占用的频率更少？

- **QAM (正交振幅调制)**：效率最高，特别是在高阶调制（如64QAM）时。
- **MFSK**：效率最低。随着进制数 $M$ 的增加，MFSK 的带宽会变**大**（更浪费），而 MPSK 的带宽不变 。

#### (3) 高阶调制对比：64QAM vs 64PSK

- **结论**：**64QAM 优于 64PSK** 。
- **原因**：64QAM 的星座点是方阵分布，64PSK 是圆周分布。在功率相同时，QAM 的点与点之间的**距离（欧氏距离）更大**，所以更不容易把 A 误判成 B，抗噪声性能更好 。
- **缺点**：QAM 设备更复杂，且对**幅度非线性**敏感 。

### 3. 复习要点总结（背这几句）

1. **想省带宽？** 模拟选 **SSB**，数字选 **QAM** 或 **PSK**。
2. **想抗干扰？** 模拟选 **FM**，数字选 **PSK**（最差是 ASK）。
3. **什么是“非线性调制”？** **FM** 和 **PM**（角度调制）。
4. **为什么用 QAM 不用 PSK？** 当 $M$ 很大（如64）时，QAM 误码率更低 。
