# 燕山大学 计算机考研详档（2026版）

## 1. 基本信息

- 地区：河北秦皇岛
- 院校层次：省属重点高校 / 机械特色高校
- 官方研招入口：https://zsjyb.ysu.edu.cn/
- 目标学院（计算机相关）：信息科学与工程学院

## 2. 招生专业与初试科目

**招生规模：** 计算机科学与技术约69人

## 3. 复试与录取规则

- 复试形式：现场/线上结合
  - 专业课笔试 + 综合面试
- 复试占比：初试50%-60%，复试40%-50%
- 同等力学加试要求：按学校要求执行
- 调剂政策：执行国家A类线

## 4. 导师与研究方向

- 导师检索入口：https://zsjyb.ysu.edu.cn/
- 代表研究方向：计算机软件与理论、计算机应用技术、人工智能

## 5. 学费与奖助体系

- 学费标准（全日制）：8000元/生·学年
- 国家助学金：6000元/生·学年（全日制非定向）
- 学业奖学金：全覆盖
- 其他专项资助：三助岗位、国家奖学金
