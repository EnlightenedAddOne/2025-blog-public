# 第一章：绪论

## 1. 数据结构的基本概念

- **数据结构三要素**：
  1. **逻辑结构**：
     - 线性结构（线性表、栈、队列、串）
     - 非线性结构（树、图、集合）
  2. **存储结构（物理结构）**：
     - 顺序存储（随机存取，占用连续空间）
     - 链式存储（指针链接，非连续）
     - 索引存储（建立索引表）
     - 散列存储（Hash，通过关键字计算地址）
  3. **数据的运算**：定义在逻辑结构上，实现在存储结构上。

## 2. 算法评价（核心考点）

### 时间复杂度 $T(n)$

- **定义**：算法中所有语句的频度之和，记为 $T(n) = O(f(n))$。
- **常见阶数排序**：
  $$O(1) < O(\log_2 n) < O(n) < O(n \log_2 n) < O(n^2) < O(n^3) < O(2^n) < O(n!)$$
- **计算技巧**：
  - 循环主体：找最深层循环的执行次数。
  - 递归算法：使用递归公式或递归树推导。
  - _例子_：`i=1; while(i<=n) i=i*2;` $\rightarrow 2^x \leq n \rightarrow x = \log_2 n \rightarrow O(\log_2 n)$

### 空间复杂度 $S(n)$

- **定义**：算法运行过程中所使用的辅助空间大小。
- **考点**：
  - 原地工作：$O(1)$。
  - 递归栈空间：深度为 $k$ 的递归调用通常需要 $O(k)$ 的空间。
  - 数组/矩阵辅助空间。
