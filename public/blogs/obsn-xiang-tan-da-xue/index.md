# 湘潭大学 计算机考研详档（2026版）

## 1. 基本信息

- 地区：湖南湘潭
- 院校层次：省属重点高校 / 双一流建设高校
- 官方研招入口：https://yz.xtu.edu.cn/
- 目标学院（计算机相关）：计算机学院·网络空间安全学院

**招生规模：** 计算机科学与技术约34人

## 3. 复试与录取规则

- 复试形式：现场/线上结合
  - 专业课笔试 + 综合面试
- 复试占比：初试50%-60%，复试40%-50%
- 同等力学加试要求：按学校要求执行
- 调剂政策：执行国家A类线

## 4. 导师与研究方向

- 导师检索入口：https://yz.xtu.edu.cn/
- 代表研究方向：计算机软件与理论、计算机应用技术、网络空间安全

## 5. 学费与奖助体系

- 学费标准（全日制）：8000元/生·学年
- 国家助学金：6000元/生·学年（全日制非定向）
- 学业奖学金：全覆盖
- 其他专项资助：三助岗位、国家奖学金
